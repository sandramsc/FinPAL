## How to use the app

**Step #1** - Clone the project

```bash
$ git clone https://github.com/sandramsc/FinPAL.git
```

**Step #2**

- Install dependencies in both frontend and backend folders: `npm i` OR `yarn`

- To start the webapp, run: `npm start` OR `yarn start`
